## 💡 使用快速开始

### 1. 库集成

```rust
use xqpath::{query, exists, XQPathDebugger};

let data = r#"{"users": [{"name": "Alice"}]}"#;
let names = query!(data, "users[*].name").unwrap();
```

### 2. 命令行使用

```bash
echo '{"users": [{"name": "Alice"}]}' | xqpath "users[*].name"
```

### 3. 交互式调试

```bash
cargo run --bin xqpath -- --debug
# 进入交互式调试模式
```
