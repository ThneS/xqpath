## 🔧 架构成就

### 模块化设计

```
src/
├── core/              # 核心功能
├── parser/            # 路径解析器
├── value/             # 值处理
├── debug/             # 调试功能 (v1.4.1+)
├── config/            # 配置管理 (v1.4.3+)
├── debugger/          # 交互式调试器 (v1.4.3+)
└── cli.rs             # 命令行接口
```
